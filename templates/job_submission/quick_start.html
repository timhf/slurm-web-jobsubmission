Quick tutorial

How to create a job

  • Click "Create a new job"
  • Enter a job name
  • Specifiy your CPU, GPU and RAM requirements
  • Pick an image. These referr to docker images available on the system. If you need a special configuration ask your supervisor to deploy it.
  • Upload your project as zip archive
  • Click next
  • Select your start script from the python image

Extending an available image

We offer images for all common frameworks and environemnts. However, if you need some extra python packages or libraries you can add them by yourself. Since every job runs in a docker container all changes are allowed, but they are not persitent in the image itself.

Instead of using, for example a python script to start your project, you can use a shell script and install required packages before starting your job. Use apt to install packages, containers are based on Ubuntu and conda for python packages. Like this:

#!/bin/bash

apt-get install ubuntu-package
conda install python-package

python your_script.py

In order to successfully run this please select either "ubuntu_raw" or "conda_raw" as base image.
